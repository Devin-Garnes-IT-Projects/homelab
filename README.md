# Homelab

Enterprise Infrastructure built with Proxmox.

## About
This repository documents my homelab journey and infrastructure projects built around Proxmox VE.

## Purpose
The goal of this homelab is to build a reliable, scalable, and production-like virtualized environment using Proxmox as the primary hypervisor. This serves as both a learning platform and a portfolio of my infrastructure work.

## Current Infrastructure

### Proxmox Host
- Hypervisor: Proxmox Virtual Environment
- Hardware: DELL INSPIRON 7640 

### Current Status

- Basic virtual machine management in progress
